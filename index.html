02·5Trajectory

Fourteen years, mostly on the GPU side of the web.

  1. 2011Frontend Engineer
    First taste of canvas.

    Built charting and data-viz for an early analytics startup. Learned that 60fps is a promise, not a goal.

  2. 2015Graphics Engineer · Luma Labs
    Went real-time.

    Owned the WebGL rendering core for a browser-based 3D editor. Cut frame time 3× with instancing and a custom render graph.

  3. 2019Staff Engineer · Northwind
    Scale & open source.

    Led the real-time graphics platform behind a product used by 40M people a month. Open-sourced the shader toolchain it was built on.

  4. 2023W3C Graphics CG · Speaker
    Gave back.

    Presented on deterministic WebGL pipelines and offered the reference implementation the group adopted for testing.

  5. NowIndependent · Principal
    Building in the open.

    Small, sharp tools and experiments — Three.js, WebGL and the occasional static-site obsession. Available for the right problem.
